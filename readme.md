# Big data management project
## Installation
1. Install sqlite
1. Create DB:
```bash
sqlite3 medical_data.db
.database
```
3. Lancer src/gui.py

## Purposes of each dataset
- DrugBank: contains several drugs, their properties and interactions with other drugs. It is used to analyze drug interactions and properties.
- SIDER: contains information on marketed drugs and their recorded adverse drug reactions.
- HPO: contains phenotypic abnormalities encountered in human disease. It is used to analyze phenotypic abnormalities and their associations with diseases.
- OMIM: contains information on human genes and genetic disorders. It is used to analyze genetic disorders and their associations with genes.
